# Примеры HTML для админ-рассылки

**Где хранится «настоящий» шаблон:** в таблице `email_templates` — создаётся и редактируется в интерфейсе **Admin → Email** после применения миграции с почтовыми таблицами.

**Зачем эта папка:** готовые фрагменты (тема + HTML), которые можно скопировать в форму создания шаблона. Письма уходят через Resend; подстановки — см. `@entities/email-template` (`{{name}}`, `{{email}}`, …).
